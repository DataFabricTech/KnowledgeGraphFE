# Demo App

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

[http://localhost:3000](http://localhost:3000)으로 어플리케이션을 확인하세요.

## 서버 연동

.env.local의 NEXT_PUBLIC_SERVER_IP 변수에 서버 IP를 입력하여 서버를 실행해 주세요. 해당 변수에 값이 할당되지 않는 경우 local 데이터를 이용하여 그래프를 실행합니다.
